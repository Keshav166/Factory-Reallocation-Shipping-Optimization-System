# Factory Reallocation & Shipping Optimization System

## Project Overview
This project helps Nassau Candy Distributor improve shipping efficiency by recommending better factory allocations for products.

## Features
- Data Cleaning and Preprocessing
- Lead Time Analysis
- Product Clustering
- Factory Reallocation Recommendations
- Streamlit Dashboard
- What-If Scenario Analysis

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Streamlit

## How to Run

pip install -r requirements.txt

streamlit run nassau_app.py
